#Redis 

## 安装

```
brow install redis
```
安装完成后运行

```
redis-server
```
在运行
```
redis-cli
```

## 使用
### 添加
设置 时候就是键值对
eg:
```
set myname xxx
```
### 查找
通过键查找值
```
get myname
```
键查找
```
keys *
```

### 删除
```
del myname
```
